# Contrast

`<Contrast>` er en code-only layout primitive; den er polymorf og støtter CSS-props som håndterer layout. Den returnerer en seksjon som inverterer tekstfargen (inkludert lenker), slik at kontrasten er riktig på invertert bakgrunn ut av boksen.
På denne måten slipper konsumenten å tenke på fargene, og komponenten følger WCAG krav samt fargesystemet til Elsa.

`<Contrast>` bruker CSS-variabler for å styre farger internt i Elsa komponentene, og CSS `color` regel for å sette farge ellers. Hvis du bruker elementer med egen `color`-styling kan det hende fargen ikke blir satt riktig.

## Props

Komponenten har default bakgrunnsfarge som kan overskrives med egen CSS. **OBS!** bruk farger med `surface-inverse` i navnet.

## Retningslinjer

- Bruk farger med `surface-inverse` i navnet for bakgrunn.
- Helst bruk Elsa-komponenter for riktig styring av tekstfarge.
- Hvis du ikke bruker Elsa-komponenter, bruk farger med `text-on-inverse` i navnet for tekstfarge.
