Tabeller bygges med styled versjoner av native HTML elementer. Disse komponentene er:

- **`<Table.Wrapper>`** - wrapper for tabell. Håndterer scrolling.
- **`<Table>`** - selve tabellen.
- **Delkomponenter:** `<Table.Head>`, `<Table.Body>`, `<Table.Foot>`, `<Table.Row>`, `<Table.Cell>`, `<Table.SortCell>`.

Enkelte delkomponenter har props for å sørge for riktig styling og oppførsel, mens andre har kun native props.

Ikke-compound komponenter er også støttet: `<TableWrapper>`, `<TableHead>`, `<TableBody>`, `<TableFoot>`, `<TableRow>`, `<TableCell>`, `<TableSortCell>`.

## Retningslinjer

### Footer

Bruk `<Table.Foot>` som footer i tabellen ved summering av verdier i en kolonne eller lignende.
