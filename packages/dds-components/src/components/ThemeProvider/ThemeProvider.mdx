`<ThemeProvider>` setter riktig tema for applikasjonen din. Den wrapper applikasjonen i en `<div>` som setter riktige CSS klasser for både Elsa komponenter og din bruk av tokens i CSS eller SCSS.
I tillegg velger den sett med tokens i TS om du skulle brukt de.

Komponenten er obligatorisk å bruke.

### Hvis du bruker tokens i TS

<Source code={`
import { ThemeProvider, useTheme } from '@norges-domstoler/dds-components';

<ThemeProvider>
  <Panel />
</ThemeProvider>

function Panel(props) {
const { theme } = useTheme();

const style = {
borderRadius: theme.ddsBorderRadiusSurface,
};

return (

<div style={style} {...props} />
)}

`} />
