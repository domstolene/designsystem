## Bruk som lenke

### Vanlig

Bruk `as` prop slik at komponenten kan returnere vanlig lenke.

<Source code={`

<Button as='a' href="/saker">
  Mine saker
</Button>
`} />

### Klient-side ruting

Bruk `as` prop slik at komponenten kan returnere lenke fra rammeverket du bruker.

#### `react-router`

<Source code={`
import { Link } from "react-router";

<Button as={Link} to="/saker">
  Mine saker
</Button>
`} />

#### `Next.js`

<Source code={`
import NextLink from 'next/link';

<Button as={NextLink} href="/saker">
  Mine saker
</Button>
`} />

## Retningslinjer

- Unngå å bruke `disabled`-attributtet da det fører til [dårlig brukeropplevelse](https://axesslab.com/disabled-buttons-suck/). Knappen skal være interaktiv, og dersom knappehandlingen ikke kan fullføres, bør det vises en tydelig feilmelding eller en annen relevant tilbakemelding.
