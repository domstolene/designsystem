Følgende komponenter er tilgjengelige:

- **`<RadioButton>`** - en radioknapp.
- **`<RadioButtonGroup>`** - en gruppe-wrapper for flere radioknapper.

## Kontrollert

<Source
  code={`
      import { RadioButton, RadioButtonGroup } from '@norges-domstoler/dds-components'
      
      const [value, setValue] = useState<number | undefined>();
   
      <RadioButtonGroup
        value={value}
        onChange={(_event, value) => {
          setValue(value);
        }}
      >
        <RadioButton value={1} label="Option 1" />
        <RadioButton value={2} label="Option 2" />
      </RadioButtonGroup>
`}
/>
