## Klient-side ruting

Bruk `as` prop slik at komponenten kan returnere lenke fra rammeverket du bruker.

### `react-router`

<Source code={`
import { Link as RRLink } from "react-router";
import { BackLink } from '@norges-domstoler/dds-components';

<BackLink as={RRLink} to="/saker">
  Mine saker
</BackLink>
`} />

### `Next.js`

<Source code={`
import NextLink from 'next/link';
import { BackLink } from '@norges-domstoler/dds-components';

<BackLink as={NextLink} href="/saker">
  Mine saker
</BackLink>
`} />
