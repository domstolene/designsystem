## Klient-side ruting

Bruk `as` prop slik at komponenten kan returnere lenke fra rammeverket du bruker.

### `react-router`

<Source code={`
import { Link as RRLink } from "react-router";
import { Link } from '@norges-domstoler/dds-components';

<Link as={RRLink} to="/saker">
  Mine saker
</Link>
`} />

### `Next.js`

<Source code={`
import NextLink from 'next/link';
import { Link } from '@norges-domstoler/dds-components';

<Link as={NextLink} href="/saker">
  Mine saker
</Link>
`} />

## Retningslinjer

- En lenke må ha `href` prop for assisterende teknologi skal tolke elementet riktig.
- Komponenten brukes kun til sematiske lenker, likke knapper el.l.. som ser ut som lenke.
