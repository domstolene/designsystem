# DdsProvider

`<DdsProvider>` setter følgende i applikasjonen din:

- Visuelt tema. Den wrapper applikasjonen i en `<div>` som setter riktige CSS klasser for både Elsa komponenter og din bruk av tokens i CSS eller SCSS. I tillegg velger den sett med tokens i TS om du skulle brukt de.
- Språk internt i komponenter. Den håndterer statiske tekster, typisk usynlige tekster for skjermeleser, men også noen synlige, som "Vis 10 av 100" i `<Pagination>`-komponenten.
- Rotelement for portaler. Brukes av `<Drawer>`, `<Modal>` osv.

Komponenten blir obligatorisk å bruke i neste major release.
