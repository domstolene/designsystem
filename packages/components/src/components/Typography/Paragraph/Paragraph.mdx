## Oversikt

`<Paragraph />` er en komponent som tilsvarer `<p>` HTML tag. Den brukes til avsnitt, ingress o.l. Den bruker oversikten over typografistiler definert i <Link external href="https://design.domstol.no/987b33f71/p/63b944-typografi/b/21516f">Typografi | Stiler og bruk</Link> til å gi tekst riktig styling.

## API

<ArgTypes story={PRIMARY_STORY} />

I tillegg støttes native HTML-attributter som er en del av
`HTMLAttributes<HTMLParagraphElement>`-interface.
`style`-prop kan settes både på toppnivå og i `htmlProps`, og blir slått
sammen av komponenten.
