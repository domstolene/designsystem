## Oversikt

`<Heading />` er en komponent som tilsvarer `<h1>`-`<h6>` HTML tag. Den bruker oversikten over typografistiler definert i <Link external href="https://design.domstol.no/987b33f71/p/63b944-typografi/b/21516f">Typografi | Stiler og bruk</Link> til å gi tekst riktig styling.

## API

I tillegg støttes native HTML-attributter som er en del av
`HTMLAttributes<HTMLHeadingElement>`-interface.
`style`-prop kan settes både på toppnivå og i `htmlProps`, og blir slått
sammen av komponenten.
