## Oversikt

Komponenten består av to subkomponenter:

- `<Breadcrumbs />` som er selve brødsmulestien
- `<Breadcrumb />` som er barna i brødsmulestien

Siste `<Breadcrumb />` skal være den siden brukeren er på, uten oppgitt URL.

## Bruk i koden

<Source code={`
import { Breadcrumb, Breadcrumbs } from '@norges-domstoler/dds-components';

<Breadcrumbs>

## API

### Breadcrumbs

Selve brødsmulestien. Skal ha `<Breadcrumb />` som barn.

<ArgsTable story={PRIMARY_STORY} />

I tillegg støttes alle native HTML-attributter som er en del av `HTMLAttributes<HTMLElement>`-interface i `htmlProps`.

### Breadcrumb

Brødsmule. Returnerer `<a>` hvis `href`-prop er oppgitt eller `<span>` hvis ikke.
**OBS!** Alle `<Breadcrumb>`-komponenter bør være lenker bortsett fra den siste som tilsvarer siden brukeren er inne på.

Det støttes alle native HTML-attributter som er en del av `AnchorHTMLAttributes<HTMLAnchorElement>` eller `HTMLAttributes<HTMLSpanElement>` -interface.

## Retningslinjer

- Helst unngå at brødsmulene legger seg over flere linjer. Test på mindre skjermer.
- Ikke bruk sesjonshistorikken i brødsmulene.
- Den siste brødsmulen skal tilsvare nåværende siden og bør dermed ikke være en lenke. Det er valgfritt å bruke `aria-current="page"` på den.
