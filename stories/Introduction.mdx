# Elsa Storybook

Velkommen til Elsa Storybook! Her kan du interagere med demoer av alle UI komponentene som ligger i domstolenes designsystem - Elsa.

Elsa Storybook kan brukes til å finne informasjon om komponentene samt testing og utvikling i isolasjon.
Elsa Storybook er også integrert med designsystemets [**dokumentasjon**](https://design.domstol.no/) - sjekk den ut for mer info om både komponentene og Elsa.
