Type `TextColor` for prop `color` i `<Typography>` og `<Spinner>` støtter ikke verdier for base fargetokens som `'interactive'` og `'gray9'` lenger. Istedet bruker vi semantiske fargetokens for tekst og ikon. Se [migreringsguiden](https://design.domstol.no/987b33f71/p/97ffbb-v15-til-v16) for detaljer. Standard verdier for `color` CSS property er støttet som før.
