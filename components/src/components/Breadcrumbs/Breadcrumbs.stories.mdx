## Oversikt

Komponenten består av to subkomponenter:

- `<Breadcrumbs />` som er selve brødsmulestien
- `<Breadcrumb />` som er barna i brødsmulestien

Siste `<Breadcrumb />` skal være den siden brukeren er på, uten oppgitt URL.

## Bruk i koden

<Source code={`
import { Breadcrumb, Breadcrumbs } from '@norges-domstoler/dds-components';

<Breadcrumbs>
  <Breadcrumb href="#">Side 1</Breadcrumb>
  <Breadcrumb href="#">Side 2</Breadcrumb>
  <Breadcrumb>Side 3</Breadcrumb>
</Breadcrumbs>
`} />

## API

### Breadcrumbs

Selve brødsmulestien. Skal ha `<Breadcrumb />` som barn.

<ArgsTable story={PRIMARY_STORY} />

I tillegg støttes alle native HTML-attributter som er en del av `HTMLAttributes<HTMLElement>`-interface.

### Breadcrumb

Brødsmule. Er en lenke eller et tekstelement. **OBS!** Alle Breadcrumb-komponenter bør være lenker bortsett fra den siste som tilsvarer siden brukeren er inne på.

<ArgsTable of={Breadcrumb} />

I tillegg støttes alle native HTML-attributter som er en del av `AnchorHTMLAttributes<HTMLAnchorElement>`-interface.
