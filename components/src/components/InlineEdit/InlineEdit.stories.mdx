## Oversikt

`<InlineEdit />` er en wrapper som håndterer inline redigering, f.eks i tabeller. Den skal ha ett barn, en av følgende:

- `<InlineEdit.Input />` - returnerer `<input>`.
- `<InlineEdit.TextArea />` - returnerer `<textarea>`.

Konsumenten knytter `<InlineEdit />` til barnet ved hjelp av `useRef()`.

## Bruk i koden

<Source code={`import { InlineEdit } fra '@norges-domstoler/dds-components';

const [value, setValue] = useState('');
const ref = useRef<HTMLInputElement>(null);

<InlineEdit {...args} inputRef={ref} value={value} onSetValue={setValue}>
  <InlineEdit.Input ref={ref} />
</InlineEdit>
`} />

## API

### InlineEdit

Wrapper for inputfeltet. Håndterer oppførselen til inputfeltet og lagring av verdien.

<ArgsTable story={PRIMARY_STORY} />
I tillegg støttes `value` fra `InputHTMLAttributes<HTMLInputElement>`-interface.

### InlineEdit.Input

Inputfeltet. Returnerer `<input>`.

<ArgsTable of={InlineEdit.Input} />
I tillegg støttes alle native HTML-attributter som er en del av `InputHTMLAttributes<HTMLInputElement>`-interface.
**OBS!** `value`, `onFocus`, `onChange`, `onBlur` settes av forelder.

### InlineEdit.TextArea

Inputfeltet. Returnerer `<textarea>`.

<ArgsTable of={InlineEdit.TextArea} />
I tillegg støttes alle native HTML-attributter som er en del av `TextareaHTMLAttributes<HTMLTextAreaElement>`-interface.
**OBS!** `value`, `onFocus`, `onChange`, `onBlur` settes av forelder.
