## Oversikt

Komponenten består av flere subkomponenter:

- `<Stepper />` - ytterste container for andre subkomponenter. Håndterer oppførsel og eventuelt litt styling. Har to eller flere `<Step />` som barn.
- `<Step />` - et steg.

## Bruk i koden

<Source
  code={`import { Stepper, Step } from '@norges-domstoler/dds-components'

export const StepperExample = () => {
  const numSteps = 3;

  const [activeStep, setActiveStep] = useState(0);
  const [completedSteps, setCompletedSteps] = useState(new Set());

  const handleStepClick = (index: number) => {
    console.log(index);
    setActiveStep(index);
  };

  const allStepsFinished = completedSteps.size === numSteps;

  return (
    <StoryTemplate title="Stepper - overview" display="block">
      <Stepper activeStep={activeStep}>
        <Step completed={completedSteps.has(0)} onClick={handleStepClick}>
          Partopplysninger
        </Step>
        <Step completed={completedSteps.has(1)} onClick={handleStepClick}>
          Slutning
        </Step>
        <Step completed={completedSteps.has(2)} onClick={handleStepClick}>
          Vedlegg
        </Step>
      </Stepper>

      {allStepsFinished ? (
        <div>Ferdig!</div>
      ) : (
        <>
          {activeStep === 0 && <div>Steg 1</div>}
          {activeStep === 1 && <div>Steg 2</div>}
          {activeStep === 2 && <div>Steg 3</div>}
        </>
      )}

      <Button
        onClick={() => {
          setCompletedSteps(s => new Set([...s, activeStep]));
          if (activeStep < numSteps - 1) {
            setActiveStep(s => s + 1);
          }
        }}
        label={allStepsFinished ? 'Du er ferdig' : 'Sett som ferdig'}
      />
    </StoryTemplate>
  );
};
`}
/>

## API

### Stepper

Ytterste container som styrer hvilket steg skal vises via indeks. Har to eller flere `<Step />` som barn.

<ArgsTable story={PRIMARY_STORY} />

I tillegg støttes alle native HTML-attributter som er en del av `HTMLAttributes<HTMLDivElement>`-interface i `htmlProps`.

### Step

Et steg.

<ArgsTable of={Step} />

I tillegg støttes alle native HTML-attributter som er en del av `ButtonHTMLAttributes<HTMLButtonElement>`-interface i `htmlProps`.
