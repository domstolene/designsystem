## API

### List

Tilsvarer `<ul>` eller `<ol>`.

<ArgsTable story={PRIMARY_STORY} />
I tillegg støttes alle native HTML-attributter som er en del av `HTMLAttributes<HTMLOListElement>` og `HTMLAttributes<HTMLUListElement>`-interface.

### ListItem

Tilsvarer `<li>`.

Det støttes alle native HTML-attributter som er en del av `HTMLAttributes<HTMLLiElement>`-interface.
