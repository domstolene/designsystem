## API

I tillegg støttes native HTML-attributter som er en del av `HTMLAttributes<HTMLElement>`-interface, `AnchorHTMLAttributes<HTMLAnchorElement>`-interface eller `LabelHTMLAttributes<HTMLLabelElement>`-interface.

### Med styled-components

Hvis du bruker styled components til å style `<Typography />` videre og skal bruke `as` prop, bruk `forwardedAs` prop istedenfor:

<Source code={`
import { ddsBaseTokens } from '@norges-domstoler/dds-design-tokens';
import { Typography } from '@norges-domstoler/dds-components';

const Label = styled(Typography)\` padding-left: \${ddsBaseTokens.spacing.SizesDdsSpacingLocalX05};\`;

export const StyledLabel = ({ label }) => {

return (

<Label forwardedAs="span" typographyType="supportingStyleLabel01">
  {label}
</Label>
); }; `} />

### Bruk av typografi-tokens

Det er også mulig å bruke typografi-tokens direkte. **OBS!** dette bør gjøres kun hvis det er umulig å bruke Typography-komponenten. Se [Bruk av font tokens i tekster med typografisk hierarki](https://zeroheight.com/987b33f71/p/18bd6f-design-tokens).
