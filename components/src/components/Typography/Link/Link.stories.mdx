## Introduksjon

`<Link />` er en komponent som tilsvarer `<a>` HTML tag. Den arver font-relatert styling. Når arvet styling skal overskrived brukes oversikten over typografistiler definert i <Link external href='https://design.domstol.no/987b33f71/p/63b944-typografi/b/21516f'>Typografi | Stiler og bruk</Link> (se API).

## API

<ArgsTable story={PRIMARY_STORY} />

I tillegg støttes native HTML-attributter som er en del av
`AnchorHTMLAttributes<HTMLAnchorElement>`-interface.
`style`-prop kan settes både på toppnivå og i `htmlProps`, og blir slått
sammen av komponenten. `onClick`, `href` og `target` støttes på toppnivå.

## Retningslinjer

- En lenke trenger ha `href` prop for assisterende teknologi skal tolke elementet riktig.
